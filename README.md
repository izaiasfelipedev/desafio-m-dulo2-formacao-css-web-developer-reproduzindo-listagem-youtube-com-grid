# Reproduzindo a Listagem do YouTube com Grid Layout no CSS

Esse segundo desafio do módulo 2 da **Formação CSS Web Developer**, consiste em aplicar a propriedade **grid**, considerando o layout da ***página inicial do YouTube***, observando a disposição dos elementos.



# Arquivos

Os arquivos está organizados seguindo a estrutura: 

## CSS 

O projeto possui dois arquivos CSS, um com as configurações globais da página HTML, com o caminho : 

**global-style.css :** /src/styles/global-style.css;

Já o outro possui as configurações principais da página:

**main.css :** /src/styles/main.css;

## imagens

As imagens utilizadas no projeto estão localizadas dentro da página imagens conforme o caminho:

**images :** /src/images;

## HTML 
O arquivo HTML está na raiz do projeto:
**index.html :** /index.html;
